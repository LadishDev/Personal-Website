## Overview

Nginx Proxy Manager provides an easy-to-use interface for managing reverse proxies and SSL certificates.

## Installation

```yaml
version: '3.8'
services:
  nginx-proxy:
    image: 'jc21/nginx-proxy-manager:latest'
    restart: unless-stopped
    ports:
      - '80:80'
      - '81:81'
      - '443:443'
```
